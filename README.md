# API Comunipsum #

## Dev ##

Build Docker
```
$ docker-compose build
```

Run Docker
```
$ docker-compose up -d
```

URL
```
http://localhost:8000
```

## Swagger ##

With the container up
```
http://localhost/documentation
```
